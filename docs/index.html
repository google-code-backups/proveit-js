Features

ProveIt is a FireFox plug-in which supports managing references on MediaWiki.

Known bugs

  • It is impossible to change tabs without the plug-in losing state. This means that if you are in the process of adding or editing a reference, you can not open or change to a new tab. To work around this, use only new windows.
  • After the sidebar first loads, clicking a reference will cause it to reload. Simply ignore this and click again after the reload.
  • Popups are partially obscured in some cases.
  • Somewhat inconsistent appearance when adding custom rows to a new reference.
