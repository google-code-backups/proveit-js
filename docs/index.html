Features

ProveIt is a Mozilla Firefox add-on for managing references on MediaWiki-based websites, including Wikipedia. It supports:

  • Editing or adding references with a convenient GUI.
  • Parsing and displaying {{cite}} and {{Citation}} style <ref>s
  • Writing generated or modified references to the edit box.
  • Inserting instances of references to any point in the document.

Known bugs

  • It is not yet possible to change tabs without the plug-in losing state. This means that if you are in the process of adding or editing a reference, you can not open or change to a new tab. To work around this, use only new windows.
  • After the sidebar first loads, clicking a reference will cause it to reload. Simply ignore this and click again after the reload.
  • Popups are partially obscured in some cases.
  • Somewhat inconsistent appearance when adding custom rows to a new reference.
