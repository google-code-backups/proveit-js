Adding a new reference

  1. Go to the page you wish to edit, and click the edit tab up top.

  2. The regular MediaWiki edit page will appear, along with the ProveIt sidebar at left. At the bottom of the sidebar, you should see {{cite}} in bold, indicating that is the selected reference type.

  3. Now, click Add a New Reference in the ProveIt sidebar.

  4. A popup will appear, showing a menu of reference types. Click the menu.

  5. For this example, we will be using a book about Borobudur. Open a new window and navigate to Borobudur: Golden Tales of the Buddhas by John Miksic, Marcello Tranchini, Anita Tranchini.

    Copy the relevant information from the reference window (in this case the Google Books page) to the ProveIt popup. You do not have to fill out all the fields. When you have copied all the information, position the caret (text cursor) at the desired position in the MediaWiki edit box. Then, click Insert in the popup.

  6. The text of the new reference will be generated, pasted into the MediaWiki edit box at the position you selected, and highlighted. In addition, since the reference is tracked by ProveIt, it will appear in the sidebar list as well, also highlighted.
