User guide

Labelled interface

  1. Edit reference button - This appears next to every reference. When you click it, the ProveIt edit popup appears so you can easily modify the reference parameters.
  2. Insert reference button - Inserts the reference at the caret (text cursor)
  3. Incomplete reference - Red text indicates a reference is missing a required parameter or otherwise invalid.

  4. Markup toggle - This toggles the kind of markup generated when a new instance of a reference is inserted. When full citation is selected, the complete reference will be outputted. An example would be:
    <ref name="Smith">{{cite book|last=Smith|First=John|Adventures with an anonymous name}}</ref>

    If name only is selected, a shortened form will be outputted, such as:
    <ref name="Smith" />

    The second is generally preferable (it eliminates redundancy in the wikisource). However, if a <ref name> was not used originally (it is optional by highly recommended), the second form can not be used. That means you must use the full form, or first add a name by editing the reference (you may use the ProveIt edit popup for this).

  5. New reference button - This launches the new referenc popup, allowing you to add a new reference.
  6. Tag style toggle - This toggles between Wikipedia's two main forms of reference tags, {{cite}} (e.g. {{cite web}}, {{cite news}}, etc.) and {{Citation}}.
  7. Edit reference popup - This is the edit reference popup. It is the GUI for adding, changing, or removing fields from a reference.
  8. Remove field - This will remove the adjacent reference parameter.
  9. Add field - This will add an additional blank field. You then fill in both the field name and value.
  10. Apply changes - This applies your edit to the reference in the MediaWiki edit box. However, the changes will only be saved to the wiki servers when you manually click the Save button below the MediaWiki text box.
  11. Cancel edit - This cancels your edit without applying any changes.
  12. New reference popup - This is the new reference popup. It is the GUI for adding a new reference.
  13. Reference type menu - This allows you to select the available reference types. This determines the available fields, and in the case of {{cite}}, which tag is generated (e.g. Web Site means {{cite web}}).
  14. Remove field - This will remove the adjacent reference parameter.
  15. Add field - This will add an additional blank field. You then fill in both the field name and value.
  16. Insert new reference - This will insert the new reference at the caret, as well as causing it to be added to the list in the sidebar.
  17. Cancel new reference - This exits the dialog without adding a new reference.
